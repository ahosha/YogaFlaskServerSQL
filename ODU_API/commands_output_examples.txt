


1)AdvancedAlignmentScanAzimuthParams:  (Same for elevation: AdvancedAlignmentScanElevationParams)
    Description:    Advanced Alignment azimuth parameters: Min Angle, Max Angle, Scan Resolution
    Attr:           CMSMAdvancedAlignmentScanAzimuthParams_ATTR_ID 36864,1916
    Parameter:      none
    Returns:        CWinAirValueSTRING
    Example:        -175,175,10 (Min,Max,Resolution)


2)AdvancedAlignmentNumOfSectorsFound
    Description:    Returns the number of sectors found in the raw alignment
    Attr:           CMSMAdvancedAlignmentNumOfSectorsFound_ATTR_ID 36864,1920
    Parameter:      none
    Returns:        CWinAirValueUINT32
    Example:        3


3)AdvancedAlignmentFullSectorsStats 
    Description:    Returns the sector's raw alignment final results
    Attr:           CMSMAdvancedAlignmentFullSectorsStats_ATTR_ID 36864,1919
    Parameter:      sector index  in range (1 - NumOfSectorsFound)
    Returns:        CWinAirValueSTRING
    Example:        1, -30, -1, 0, 0, 5780, 40000, -54, MySector, 0, 0, 0, Lat, Long

                    Sector Type (Gen3 / Gen4) (byte for future)
                    ULC Angle (short)
                    ULC Elev. (short)
                    Sector Directn.
                    Sector Ant. Beam width.
                    Channel(short)
                    CBW (byte)
                    Best RSS (byte)
                    SectorID (24 bytes)
                    Available Assured Resources DL (%)(byte)
                    Available Assured Resources UL (%)(byte)
                    Best Effort HSU can be added(bit) (1=can be added)
                    Lat
                    Long
